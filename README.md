# Sambung Kata Pro v1.0

Script untuk game **Sambung Kata** di Roblox.

## Fitur
- **Auto Type** — Otomatis mengetik dan submit jawaban
- **Kamus KBBI** — Database kata bahasa Indonesia built-in (A-Z)
- **Word Lock System** — Mencegah race condition saat mengetik
- **Reactive Interrupt** — Bisa batal ngetik jika prefix berubah
- **Human-Like Typing** — Delay acak agar natural
- **Smart Retry** — Otomatis cari kata baru jika ditolak server
- **Overlay Saran Kata** — UI draggable untuk melihat pilihan kata

## Penggunaan
1. Inject script ke Roblox executor
2. Aktifkan toggle "Auto Play"
3. Atur kecepatan mengetik sesuai keinginan
